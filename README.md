# Pill Counter

Current version: 0.0.3

This is an Android app to track medications and pills you take, allowing you to be aware of their current quantities so you avoid interrupting your treatment because you forgot to purchase more!

This is an official upgrade from the CLI Python version of this application (https://github.com/GabrielGRSilva/PC-Pill-Counter).

## INSTALATION

Just run the APK. You may need to set up the permission to install apps from APK files on your Android phone.

## USAGE

Add new meds by clicking the + button;
Add or remove individual pills (when you purchase more or take them daily) by clicking the + or - buttons on the medication list!
